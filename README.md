# poke-linebot

## 概要

- ポケモンの名前をメッセージとして送信すると、タイプ、特性、種族値を返します

- メガシンカポケモンは別ポケモンとして扱います

- フォルムチェンジ、リージョンフォームは同じポケモンとして扱います
